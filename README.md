# Business-Performance-and-Sales-Analytics-Dashboard #

This project presents an end-to-end Business and Sales Analytics solution built on a multi-year retail dataset. The goal is to uncover actionable insights that help businesses improve profitability, identify loss drivers, optimize discounts, and understand customer behavior using SQL and Power BI.
The dashboard transforms raw sales data into decision-ready visual insights for business stakeholders.
