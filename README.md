# Business Performance and Sales Analytics Dashboard

This project analyzes retail sales data to uncover revenue drivers, profitability trends, and regional performance patterns.

Tools Used:
- SQL
- Excel
- Power BI

Project Workflow:
1. Data exploration and KPI analysis using SQL 
2.Data preparation and calculations in Excel
3.Interactive dashboard creation in Power BI

Key Insights:
- Identified loss-making product categories
- Analyzed regional sales performance
- Evaluated impact of discounting on profitability

The Power BI dashboard contains interactive pages covering:
- KPI overview
- Regional performance
- Monthly sales trends
- Customer insights
