# Business-Performance-and-Sales-Analytics-Dashboard
Business performance and sales analytics dashboard of a multi year superstore dataset displaying important factors that can be helpful for  getting insights of the performance of sales and their business because of which they can improve in the overall profitability and other parameters required, by getting categorized  visualizations.

This project analyzes retail sales data to identify revenue drivers, loss-making products, regional performance, and customer insights using SQL and Power BI.

### Repository Structure
- data: Superstore Dataset(Kaggle)
- sql: All SQL queries used for analysis
- results: Query outputs exported from SQL Server
- screenshots: Dashboard and query previews

### Tools Used
SQL Server (SSMS), Excel, Power BI

### Key Analysis
- Region-wise sales and profit
- Discount impact on profitability
- Monthly sales trend
- Top 10 customers using window functions

### Outcome
Built a business dashboard to support data-driven decision making.
